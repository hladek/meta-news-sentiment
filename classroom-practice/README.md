# Classroom Practice: Metacognitive Sentiment Analysis

This folder contains a classroom activity that uses the Sentiment Analysis tool to help students monitor, test, and revise their own thinking.

## Goal

Students will increase metacognitive ability by practicing how to:

- Predict how a text will be classified before using the tool.
- State confidence levels and reasons for their predictions.
- Compare their judgment with the tool's output.
- Identify why disagreements happened.
- Revise their thinking using evidence from the text.
- Reflect on what strategies helped them think more accurately.

## Suggested Time

45 to 60 minutes.

## Tool Use

Students paste a short text into the Sentiment Analysis app, run the analysis, and review the sentence-level labels. The learning focus is not whether the tool is always correct. The focus is whether students can explain, monitor, and improve their own reasoning.
